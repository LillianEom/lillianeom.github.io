## Java概述
JVM：独立于硬件和操作系统，将字节码转换为特定机器代码，提供了内存管理/垃圾回收和安全机制等。JVM正是Java程序可以一次编写多处执行的原因。

JDK = Java语言 + JVM + Java API类库
JDK是用于支持Java程序开发的最小环境

JRE = JVM + Java SE API子集
JRE是支持Java程序运行的标准环境

Java ME：移动终端
Java SE：桌面级应用
Java EE：企业应用
